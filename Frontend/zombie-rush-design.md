# Zombie Rush Enhancement Plan

## Fantasy

Zombie Rush works best as a pressure-heavy survival arena where the player is not only trying to stay alive, but also deciding when to risk aggression for a bigger score.

## Enhanced Core Loop

1. Enter the arena and establish spacing against walkers.
2. Build a kill combo by clearing targets quickly before the chain expires.
3. Spend dash defensively to break out of swarms or cut toward drops.
4. Prioritize elite threats and collect pickups at safe moments.
5. Survive the next wave spike, then push for a higher score multiplier.

## What Changed In This Pass

- Added three enemy roles:
  - `walker`: baseline pressure unit
  - `runner`: faster flanker that punishes tunnel vision
  - `brute`: slow, tanky threat that can drop rewards
- Added a combo system that rewards aggressive wave control.
- Added a dash cooldown so movement has a clutch escape tool.
- Added pickup-driven choices:
  - `medkit`: restores health
  - `fury core`: temporary rapid-fire power spike
- Expanded the HUD so the player can read combo, boost state, and dash readiness.

## Why This Loop Is Stronger

- The player now alternates between survival and score greed.
- Enemy variety creates target priority instead of pure circle-kiting.
- Pickups add moment-to-moment routing decisions.
- Dash gives the game a skill expression layer without changing the control scheme too much.

## Recommended Next Steps

1. Add simple arena props or blockers so movement paths matter more.
2. Add a wave intro card with enemy forecasts before major spikes.
3. Add one weapon variation, such as a pierce shot or shotgun pickup.
4. Move simulation state into clearer modules if the game keeps growing.
